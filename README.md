# ARCHIVES Website

## File Structure

```
archives-site/
├── index.html      # Main single-page website markup
├── style.css       # Global styles, layout, theme, responsiveness, animations
├── script.js       # Navigation, reveal animations, filters, counters, form UX
└── README.md       # Project notes and structure
```

## Notes
- Built with HTML, CSS, and vanilla JavaScript
- Uses Google Fonts and Font Awesome via CDN
- Ready for deployment on Netlify, Vercel static hosting, GitHub Pages, or cPanel
- Contact form is frontend-only and can be connected to a live backend or form service
