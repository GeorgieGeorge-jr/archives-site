# ARCHIVES Website Refactor

This version restructures the original one-page website into a multi-page static site while preserving the main interactions.

## Pages
- `index.html` — home / brand overview
- `services.html` — services and pricing
- `portfolio.html` — portfolio with filters
- `process.html` — project workflow
- `team.html` — team and studio philosophy
- `contact.html` — contact form and FAQ

## Styles
- `css/base.css` — shared layout, components, mobile responsiveness
- `css/*.css` — page-specific styling

## Scripts
- `js/main.js` — shared navigation, reveal animations, cursor glow, tilt cards
- `js/home.js` — animated counters on the homepage
- `js/portfolio.js` — portfolio filtering
- `js/contact.js` — contact form success state

## Notes
- Mobile spacing, navigation, button stacking, card behavior, and grid collapse were improved to feel cleaner on smaller screens.
- The contact details use the latest details you provided during this project.
